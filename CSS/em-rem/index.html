em

O em pega a unidade de medida do elemento pai como referencia (tamanho padrao do html 16px)

font-size: 1em = 16px

font-size: 2em = 32px

font-size: 3em = 48px

nesse caso os textos estao dentro de uma div aonde o font-size e 25px

font-size: 1em = 25px

font-size: 2em = 50px

font-size: 3em = 75px

rem

o rem pega como base a medida do font-size do html que e 16px independente do tamanho do elemento pai (so vai ser alterado o valor se o elemento raiz html for alterado)

font-size: 1rem = 16px

font-size: 2rem = 32px

font-size: 3rem = 48px

nesse caso os textos estao dentro de uma div aonde o font-size e 25px

font-size: 1rem = 16px

font-size: 2rem = 32px

font-size: 3rem = 48px
